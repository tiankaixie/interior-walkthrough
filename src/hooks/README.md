## Architecture
Custom React hooks for deck.gl GLB viewer functionality including camera controls, model loading, and point cloud extraction.

## File Registry

| Name | Status | Core Function |
|------|--------|---------------|
| useFirstPersonControls.js | Active | WASD + mouse camera controls with physics/free-flight modes |
| useModelLoader.js | Active | GLB/GLTF model loading using loaders.gl |
| usePointCloudExtractor.js | Active | Extract point cloud data from loaded meshes |
